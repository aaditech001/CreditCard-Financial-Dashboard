# CreditCard-Financial-Dashboard
This project presents an interactive Power BI dashboard developed using SQL-cleaned credit card and customer transaction data.
The dashboard analyzes customer demographics, spending behavior, card usage patterns, and revenue performance to identify key drivers of financial growth.

~ Which spending categories (fuel, travel, bills, entertainment, etc.) drive revenue
